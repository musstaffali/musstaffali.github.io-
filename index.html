Digital Wallet

Java, MVC, Springboot, REST API, PostgreSQL

Digital Wallet

My Third Capstone at Tech Elevator! This application functions similarly to the popular digital wallet app "Venmo". It uses the MVC model, SpringBoot (Security, Web), Restful APIs, PostgreSQL, and Java!

  • Application uses User Authentication (Spring security)
  • Users can send money to other users in the system
  • Application saves all transactions to PostgreSQL database
  • Users can view all transactions between themselves and others
  • Application displays the status of transactions and amounts
  • Users can approve or decline transaction requests
  • Transactions will only occur if approved and funds are sufficient
  • https://github.com/kphanplan/Digital-Wallet
